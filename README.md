# Форум с новостями
Приложение поддерживает систему регистрации, публикации постов, комментариев в реальном времени, систему тегов.

`Frontend`

<ul>
  <li>NextJS / React</li>
  <li>TypeScript</li>
  <li>Redux / Redux Saga</li>
  <li>EditorJS</li>
  <li>React Hook Form / Yup</li>
  <li>Material UI</li>
  <li>SCSS / CSS-modules</li>
</ul>

`Backend`

<ul>
  <li>NestJS</li>
  <li>TypeScript</li>
  <li>PostgreSQL</li>
  <li>JWT / PassportJS</li>
</ul>
